Extensions

This page lists extensions that add features to Concordion. See the Extensions API for details of how to create extensions.

Alternative Run Strategies

Extension Description Maintainer
Parallel Run runs specifications in parallel when using the concordion:run command Nigel Charman

Alternative Input Sources

Extension Description Maintainer
Excel enables Concordion to use an Excel spreadsheet as an input specification Rob Moffat

Specification Enhancers

Extension Description Maintainer
Screenshot adds screenshots to Concordion output to diagnose problems or improve the documentation Nigel Charman
Storyboard captures screenshots or data in a storyboard format to provide insight into the actions a test has taken Andrew Sumner
Logging Tooltip adds logging information as tooltips to the Concordion output Nigel Charman
Logback adds Logback logging information as tooltips and as a link to the log file. Supports parallel runner Andrew Sumner
Collapse Output makes elements with a specific style class collapsible in the Concordion report Carlo M-R
Input Style adds CSS styling for input elements in the Concordion output John Hurst
Tommy Yim Li
Run Totals displays the total number of child tests passing and failing, when using the concordion:run command Tim Wright
Timestamp Formatter changes the footer of the Concordion output to show hours, minutes and seconds Nigel Charman
Exception Translator modifies the text of exception messages Nigel Charman

Additional Commands

Extension Description Maintainer
Screenshot adds screenshots to Concordion output to diagnose problems or improve the documentation Nigel Charman
Embed embeds HTML in the Concordion output Nigel Charman
Execute Only If conditionally executes child tests Andrew Sumner

Notes

The combined concordion-extensions project has been deprecated, and split into the separate projects above.

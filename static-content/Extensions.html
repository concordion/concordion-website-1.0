Extensions

This page lists extensions that add features to Concordion. See the Extensions API for details of how to create extensions.

Alternative Input Sources

  • Excel Extension — enables Concordion to use an Excel spreadsheet as an input specification

Specification Enhancers

  • Screenshot Extension — adds screenshots to Concordion output to diagnose problems or improve the documentation
  • Storyboard Extension — captures screenshots or data in a storyboard format to provide insight into the actions a test has taken
  • Logging Tooltip Extension — adds logging information as tooltips to the Concordion output
  • Logback Extension — adds Logback logging information as tooltips and as a link to the log file. Supports parallel runner
  • Collapse Output Extension — makes elements with a specific style class collapsible in the Concordion report
  • Input Style Extension — adds CSS styling for input elements in the Concordion output
  • Timestamp Formatter Extension — changes the footer of the Concordion output to show hours, minutes and seconds
  • Exception Translator Extension — modifies the text of exception messages

Additional Commands

  • Screenshot Extension — adds screenshots to Concordion output to diagnose problems or improve the documentation
  • Embed Extension — embeds HTML in the Concordion output
  • Execute Only If Extension — conditionally executes child tests

Notes

The combined concordion-extensions project has been deprecated, and split into the separate projects above.

Additional contributions are welcome. Please get in touch on the concordion-dev list.
